# 인형뽑기방 정보 서비스 기능 정의서

## 1. 지도 기반 매장 검색

### 1.1 지도에서 매장 찾기
- 카카오맵 기반(또는 네이버플레이스,  어쨋든 사업자가 본인이 등록한 거)으로 전국의 인형뽑기방 위치 확인
- 현재 위치 기준으로 주변 인형뽑기방 검색  (웹이니까 일단 킾 )
- 지역/주소로 인형뽑기방 검색
- 지도에서 매장 마커 클릭 시 간단한 정보 미리보기


###
매장 기본 정보 => 사용자가  리뷰 , 후기 남기기 
리뷰에는 사진첨부, 별점, 텍스트,    기계힘(되게 추상적이긴함),    인형 1개당 평균 지출(대형 중형 소형별로)
커뮤니티 게시판 =>  단순 후기가 아니라 블로그 식으로.   또는 어디뽑았는데 잘뽑았다 자랑 등등   커뮤니티는 좀 자유롭고,    리뷰가 정확해야지
리뷰를 기반으로 정렬 및 통계= 나중
