# 인형뽑기방 정보 서비스 기능 정의서

## 1. 지도 기반 매장 검색

### 1.1 지도에서 매장 찾기
- 카카오맵 기반으로 전국의 인형뽑기방 위치 확인
- 현재 위치 기준으로 주변 인형뽑기방 검색
- 지역/주소로 인형뽑기방 검색
- 지도에서 매장 마커 클릭 시 간단한 정보 미리보기

### 1.2 매장 목록 보기
- 검색된 인형뽑기방 목록 조회
- 거리순, 평점순, 리뷰 많은 순 정렬
- 매장 기본 정보 확인 (이름, 주소, 영업시간, 연락처)
- 단순 카카오 API정보뿐만 아니라  사용자들이 직접 방문해서 등록해야  있는걸로 할 수도 있게
  (카카오로 제공하는 기본정보 +   사용자들이 직접 방문한 실제 운영중인 가게 페이지  (여기는 자체 DB ) ) 

---



## 2. 매장 상세 정보

### 2.1 매장 정보 조회
- 매장 상세 정보 확인
- 매장 사진 갤러리 보기
- 영업시간, 주차 정보, 편의시설 확인
- 매장까지의 길찾기

### 2.2 보유 기계 정보
- 매장에 있는 인형뽑기 기계 목록 조회
- 기계별 난이도/힘 세기 정보 확인
- 기계 종류별 분류 (일반, 다리, 타워 등)

### 2.3 인형 정보
- 매장에서 뽑을 수 있는 인형 목록 조회
- 인형 종류별 분류 (캐릭터, 브랜드별)
- 인형 사진 및 상세 설명 보기
- 입고일/재고 상태 확인

---

## 3. 커뮤니티 (리뷰 & 후기)

### 3.1 매장 리뷰 작성
- 방문한 매장에 대한 리뷰 작성
- 별점 평가 (1~5점)
- 사진 첨부 가능
- 방문 날짜 기록

### 3.2 리뷰 조회
- 매장별 리뷰 목록 보기
- 최신순, 평점순, 도움순 정렬
- 리뷰 검색 (키워드)
- 사진이 포함된 리뷰만 필터링

### 3.3 리뷰 상호작용
- 다른 사용자 리뷰에 도움됨/도움안됨 평가
- 리뷰에 댓글 작성
- 내가 작성한 리뷰 수정/삭제

### 3.4 성공 인증
- 인형 뽑기 성공 인증샷 업로드
- 어떤 매장에서 어떤 인형을 뽑았는지 기록
- 성공 인증 갤러리 보기

---

## 4. 회원 기능

### 4.1 회원가입 및 로그인
- 카카오 계정으로 간편 로그인
- 구글 계정으로 간편 로그인
- 이메일/비밀번호로 직접 회원가입 및 로그인
- 비밀번호 찾기/재설정

### 4.2 프로필 관리
- 닉네임, 프로필 사진 설정
- 내 정보 수정
- 회원 탈퇴

### 4.3 마이페이지
- 내가 작성한 리뷰 목록 조회
- 내가 뽑은 인형 기록 (성공 인증 목록)
- 즐겨찾기한 매장 목록
- 관심 인형 목록

---

## 5. 매장 즐겨찾기 & 알림

### 5.1 즐겨찾기
- 자주 가는 매장 즐겨찾기 등록
- 즐겨찾기 목록에서 빠른 접근

### 5.2 관심 인형 등록
- 원하는 인형을 관심 목록에 추가
- 관심 인형이 있는 매장 알림 (선택사항)

---

## 6. 검색 기능

### 6.1 통합 검색
- 매장명으로 검색
- 인형 이름으로 검색
- 지역명으로 검색

### 6.2 필터 검색
- 조건별 매장 필터링
  - 영업 중인 매장만 보기
  - 평점 이상 매장만 보기
  - 주차 가능 매장만 보기
- 인형 필터링
  - 캐릭터별, 브랜드별
  - 신상품만 보기

---

## 7. 랭킹 & 추천

### 7.1 인기 매장
- 평점 높은 매장 TOP 10
- 리뷰 많은 매장 TOP 10
- 최근 인기 급상승 매장

### 7.2 인기 인형
- 가장 많이 뽑힌 인형 순위
- 최근 입고된 신상 인형

### 7.3 추천 시스템
- 내 위치 기반 추천 매장
- 내 관심사 기반 추천 인형
- 비슷한 취향의 사용자들이 좋아하는 매장

---

## 8. 기타 편의 기능

### 8.1 최근 본 항목
- 최근 본 매장 목록
- 최근 본 인형 목록

### 8.2 공유 기능
- 매장 정보 공유 (카카오톡, URL 복사)
- 리뷰 공유
- 성공 인증 공유

---

## 부가 설명

- **대상 사용자**: 인형뽑기를 즐기는 모든 사용자
- **주요 가치**: 매장 정보의 투명성, 커뮤니티 기반의 실시간 정보 공유
- **차별점**: 배달의민족처럼 상세한 매장별 리뷰 시스템과 실제 보유 인형/기계 정보 제공
