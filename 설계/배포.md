🏗️ 도커 배포 전체 과정 (End-to-End)
1단계: 내 컴퓨터 (제작 및 포장)가장 먼저 내 컴퓨터에서 '배포할 수 있는 상태'를 만듭니다.
빌드: Gradle이나 Maven을 이용해 프로젝트를 실행 파일(.jar)로 만듭니다.
레시피 작성: Dockerfile을 만들어 "이 자바 파일을 어떤 환경에서 실행할지" 적습니다.
박제(Build): Docker Desktop을 이용해 내 코드와 환경을 하나로 묶은 **이미지(Image)**를 생성합니다.
2단계: 도커 허브 (보관 및 운반)내 컴퓨터에만 있으면 서버가 가져갈 수 없으니, 공용 저장소에 올립니다.
업로드(Push): 내 컴퓨터에서 만든 이미지를 Docker Hub라는 인터넷 저장소로 보냅니다. 
(유튜브에 영상 올리는 것과 같습니다.)
3단계: 클라우드 서버 (실행)이제 AWS나 오라클 클라우드 같은 실제 서버에서 프로그램을 돌립니다
.준비: 서버에 도커를 설치합니다.
다운로드(Pull): 서버에서 "도커 허브에 있는 내 이미지를 가져와!"라고 명령합니다.
실행(Run): 가져온 이미지를 실행하여 **컨테이너(미니 서버)**를 띄웁니다.
접속: 이제 전 세계 어디서든 서버의 IP 주소로 내 스프링부트에 접속할 수 있게 됩니다.
